一、如何在MAYA中導入英雄的3D模型？如何將紋理貼圖綁定到模型上？

1、使用“Import”，選擇轉換好的.obj文件即可加載模型。

2、使用“Window - Rendering Editors - Hypershade” ，在左邊的目錄中選擇“Surface”，新建一個“Blnn”類型的材質球

3、指定該“Blnn”材質球的“Common Material Attributes - Color”為“File”（注意用右鍵點“File”，選“Create Texture”）

4、在“File Attributes - Image Name” 中選擇需要的紋理貼圖文件

5、按 6 即可查看~

二、如何提取英雄的3D骨骼？如何在Maya 中載入英雄的3D骨骼？

1、在Script Editor中的Python部分輸入以下代碼：

from maya import mel
import maya.cmds as cmds
def importSkeleton(fileName):
f = open(fileName,"rb")
f.readline()
f.readline()
f.readline()
numBones = int(f.readline())
bones = []
for i in range(0,numBones):
  boneName = f.readline().split()[0]
  parent = int(f.readline())
  scale = float(f.readline())
  matrix = []
  matrix.append(f.readline().split())
  matrix.append(f.readline().split())
  matrix.append(f.readline().split())
  bones.append((i,boneName,parent,scale,matrix))
for bone in bones:
  cmds.select(d=True)
  cmds.joint(p=(float(bone[4][0][3]),float(bone[4][1][3]),float(bone[4][2][3])), n=bone[1]+"_"+str(bone[0]))
for bone in bones:
  if bone[2] != -1:
   cmds.select(bone[1]+"_"+str(bone[0]))
   cmds.select(bones[bone[2]][1]+"_"+str(bones[bone[2]][0]),add=True)
   cmds.parent()

2、點選菜單中的“execute”

3、退出Script Editor，在命令行中輸入以下指令（請更改骨骼txt文件的文件名和路徑，目錄請注意使用雙斜杠）如: python "importSkeleton('C:\\Morgana\\Morgana.skl.txt' )"


三、怎樣知道模型不同區域與紋理貼圖文件不同區域的對應關係？

1、在Maya 中選好希望查看對應關係的模型，然後選擇"Window - UV Texture Editor" 即可查看，

2、如果需要生成不帶紋理的對應關係圖，請在該編輯器中選擇“Polygons - UV Snapshot...” ，尺寸請指定為512x512 或其它等比例尺寸~


四、如何導出已經在MAYA中修改完成的英雄3D模型？

1、修改好模型後（一定記得要讓模型先Triangulate），使用“Script Editor” 在Mel 部分載入“Maya_LoL_skn_export.mel”

2、然後選擇希望輸出的模型，然後輸入（記得更改成你需要的文件名字和路徑哦）： exportMesh("C:\\Path\\File.txt")

3、由於模型已經轉成.txt 文件，下面我們運行"lolskintool.exe c skn File.txt File.skn" 來得到符合遊戲標準的.skn 模型文件~

五、如何修改英雄載入畫面？如何讓載入畫面顯示正確的圖片比例？

1、創建好您希望用做載入畫面的圖片，等比例縮放到寬362個像素（高須大於570），移動、剪切調整到高570個像素

2、非等比縮放（“雙立方”方式）到307像素寬，483像素高

3、裁切畫佈到301像素寬，477像素高，將畫布底色設定為純黑，重新將畫布擴大到307像素寬，483像素高

4、從上向下擴大畫佈到307像素寬，557像素高，將文件保存為PNG 格式

3、使用The Compressonator 載入PNG文件，然後使用“Save Original...” 功能另存為英雄名稱+“LoadScreen.dds” 文件

4、放到英雄聯盟安裝目錄的ZIP裡 即可
